# 1️⃣ E-mail Profissional

Crie seu e-mail no Gmail:

👉 [https://accounts.google.com/signup](https://accounts.google.com/signup)

### Passo a passo:

1. Preencha seu **nome e sobrenome**
2. Escolha um e-mail profissional
   Exemplo:

   ```
   leonardorosolar@gmail.com
   ```
3. Crie uma senha forte:

   * Letras maiúsculas e minúsculas
   * Números
   * Símbolos
4. Confirme seu celular
5. Finalize o cadastro

💡 **Dica:**
Evite coisas como `gamer123` — isso tira sua credibilidade.

# 3️⃣ Criar seu Primeiro Site

### 📁 Crie o arquivo

1. Abra o editor
2. Crie uma pasta:

```
meu-primeiro-site
```

3. Crie o arquivo:

```
index.html
```

---

### 🤖 Gere o código com IA

Use este prompt:

```
Crie um arquivo index.html completo com o título "Meu Primeiro Site". 
O site deve ter: um cabeçalho com meu nome, uma seção "Sobre mim" com um 
parágrafo de apresentação, uma seção "Projetos" com 3 cards e um rodapé. 
Use apenas HTML puro, sem arquivos externos. Estilo visual moderno com CSS inline.
```

### Como usar:

* Cursor → `Ctrl + K`
* Ou use: [https://claude.ai](https://claude.ai)

---

### ▶️ Rodar o site

1. Salve (`Ctrl + S`)
2. Abra o arquivo no navegador

💡 **Dica:**
Você já é oficialmente um dev agora 😄

# 4️⃣ Criar Conta no GitHub

👉 [https://github.com/signup](https://github.com/signup)

### Passos:

1. Use o **mesmo e-mail profissional**
2. Escolha um username (ex: `leonardorocha`)
3. Confirme o e-mail
4. Clique em:

```
New repository
```

### Configuração:

* Nome:

```
meu-primeiro-site
```

* Público (Public)

💡 **Por quê GitHub?**
É o portfólio oficial de todo desenvolvedor.

---

# 5️⃣ Subir o Código

1. Entre no repositório
2. Clique em:

```
uploading an existing file
```

3. Envie o arquivo:

```
index.html
```

4. Commit:

```
Adiciona página inicial
```

💡 **O que é commit?**
Um “save” do seu projeto.

# 6️⃣ Publicar (GitHub Pages)

1. Vá em:

```
Settings → Pages
```

2. Em **Branch**, selecione:

```
main
```

3. Clique em **Save**

### Seu site ficará em:

```
https://seuusuario.github.io/meu-primeiro-site
```

💡 **Parabéns:**
Seu site está no ar 🚀

# 8️⃣ Conectar Domínio ao GitHub

### No Registro.br:

Adicione registros A:

```
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153
```

E um CNAME:

```
www → seuusuario.github.io
```

---

### No GitHub:

```
Settings → Pages → Custom domain
```

Exemplo:

```
leonardosolar.com.br
```

Ative:

```
Enforce HTTPS
```

⏱️ Pode levar até 24h
